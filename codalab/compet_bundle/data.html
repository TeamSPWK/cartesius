Data

In this competition, we only make the validation and the test set available.

Files

  • testset.json : The test set
  • valset.json : The validation set
  • sample_submission.json : A sample submission file, with the correct format

Test & Validation sets

These 2 files contain hand-crafted geometries, covering a wide variety of geometries, with different positions and scales.

There is one geometry per line, represented by a WKT string. You can load these geometries with shapely :

from shapely import wkt
polygon = wkt.loads(wkt_string)

Which data to use for training

This is free : use whatever you want to train your model !

The easiest way is to use cartesius package, which provides a Pytorch dataset where polygons are randomly generated.
